# Editorial Quality Gate Rubric

Score each dimension 1-5. Publish only if every dimension is >=4.

## 1) Novelty
- 5: Non-obvious insight with a clear reframing.
- 3: Useful but familiar framing.
- 1: Generic restatement.

## 2) Specificity
- 5: Concrete constraints, tradeoffs, and failure modes.
- 3: Some specifics, limited constraints.
- 1: Abstract advice.

## 3) Utility
- 5: Action checklist can be executed in 7 days.
- 3: Useful but vague next actions.
- 1: No clear execution path.

## 4) Credibility
- 5: Strong logic plus practical scenario/evidence.
- 3: Plausible but lightly supported.
- 1: Assertion-heavy.

## 5) Clarity
- 5: Single thesis, tight narrative, no drift.
- 3: Mostly clear with mild detours.
- 1: Mixed arguments.

## Pre-Publish Checks
- Reader can summarize thesis in one sentence.
- Reader can apply one action this week.
- Includes at least one explicit tradeoff and one failure mode.
- References one prior or next article in the series.
